# Stepp0r

You have a Launchpad? 
And you use Renoise? 
Or you like Step-Sequencing? 
Well you're lucky!
I wrote a Step-Sequencer using the Launchpad and Renoise.

###Features
<div class="block-feature" markdown='1'>
* Step-Sequencing
* Pattern Arrangment
* Copy Paste
</div>
